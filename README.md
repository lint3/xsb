## Use It

### Get the code

```
git clone git@github.com:lint3/xsb.git
cd xsb
```

### Set up Python venv

```
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python -V
python app.py
```

## Update It

```
git pull
source venv/bin/activate
pip install -r requirements.txt
```
